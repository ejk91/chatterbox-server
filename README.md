# Chatterbox Server
Your new favorite chat server - in raw Node.js

## Team
  - Enoch Kim
  - Neil Ricci

## Requirements

- babel
- chai 
- mocha 

## Development
### Installing Dependencies
From within the root directory:
```sh
npm install
```
### Run test suite
```sh
npm run test
```
### Start Node Server
```sh
npm run start
```
